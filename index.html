درباره ما

این یک اپلیکیشن تعاملی کودک‌پسند است که با صدای کودک تعامل می‌کند.

کودک می‌تواند با گفتن کلماتی مانند «توپ»، «فیل»، «خورشید»، «ستاره» و «گل» شاهد تغییر شکل دایره به تصویر متناسب با آن کلمه باشد.

این اپلیکیشن با استفاده از Web Speech API طراحی شده و برای همه دستگاه‌ها قابل استفاده است.
